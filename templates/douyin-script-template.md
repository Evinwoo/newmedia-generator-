# 抖音口播脚本模板

适用于短视频口播、达人种草、两人对话、产品测评、信息流广告脚本。  
核心原则：前3秒先抛痛点，中段用真实体验证明，结尾给明确行动。

## 格式一：通用脚本结构

```
3秒钩子：痛点/反常识/尴尬场景
↓
问题放大：用户为什么在意
↓
解决方案：产品自然出现
↓
体验证明：时间、状态、触感、对比
↓
卖点补充：成分、质地、价格、规格
↓
行动收口：适合谁、怎么买、囤几支
```

## 口播脚本拆解：腋下止汗净味乳霜

### 原始脚本信息

产品：宫教授止汗净味乳霜  
核心人群：腋下有味道、爱出汗、夏天穿衣怕黏腻的人  
核心场景：早上出门前涂，下午仍然干爽；夏天穿紧身衣不想黏  
核心卖点：止汗净味、清爽、吸收快、不痒不黏、持久、便宜、三支装

## 一、开头钩子拆解

### 原文

```
洗腋下是没用的，还是会有味，怎么办？赶紧涂去。（两人对话）
```

### 作用

- 用反常识开场："洗腋下是没用的"制造停顿感。
- 直接点出用户痛点：洗了还是有味。
- 用两人对话让脚本更像生活场景，不像硬广。
- "赶紧涂去"快速引出解决方案。

### 可复用公式

```
A：我都洗干净了，怎么还是有味？
B：光洗真的不够，出门前先涂这个。
```

### 可替换钩子

```
腋下有味不是你没洗干净，可能是少了出门前这一步。
```

```
夏天最尴尬的不是出汗，是洗完没多久又有味。
```

```
如果你腋下总是湿湿的，真的别只靠沐浴露。
```

## 二、场景证明拆解

### 原文

```
我今天早上涂了这个，现在下午两点了，很干爽，上面写着止汗净味，很好用。
```

### 作用

- "今天早上"和"下午两点"给出时间证据。
- "很干爽"是用户能立刻理解的结果。
- "上面写着止汗净味"把产品功能自然读出来。
- "很好用"是口语化评价，适合达人风格。

### 可复用公式

```
我今天[时间点]涂的，现在已经[时间点]了，[部位/状态]还是[具体感受]。
```

### 示例

```
我早上出门前涂的，现在下午两点了，腋下还是比较干爽。
```

## 三、人群适配拆解

### 原文

```
只要是这里有味道、爱出汗的，就每天去拿宫教授涂一涂，很清爽。
```

### 作用

- 明确适合人群：有味道、爱出汗。
- "这里"配合手势指向腋下，口播更自然。
- "每天涂一涂"降低使用门槛。
- "很清爽"重复核心体验。

### 可复用公式

```
只要你是[痛点人群]，就可以在[使用时机]拿它[动作]，用完是[感受]。
```

## 四、质地体验拆解

### 原文

```
它就是这种乳霜的质地，涂上去几秒就吸收了，涂上去也没有什么不舒服，而且也不痒。
```

### 作用

- 展示质地，适合配合手部试涂画面。
- "几秒就吸收"解决黏腻顾虑。
- "没有不舒服、不痒"降低第一次使用的心理门槛。

### 镜头建议

```
手背挤出乳霜
↓
指腹推开
↓
展示吸收后的皮肤状态
↓
口播强调：不黏、不闷、不痒
```

### 可复用公式

```
它是[质地]，涂开之后[吸收速度]，不会[用户担心的问题]。
```

## 五、持久体验拆解

### 原文

```
我觉得这个真的很牛，出门之前15分钟涂上，到晚上再次洗澡之前，这里都是没有什么味道的，因为我这里就老是湿湿的。
```

### 作用

- "真的很牛"是情绪评价，提升达人真实感。
- "出门之前15分钟"给出具体使用方法。
- "到晚上再次洗澡之前"强化持久感。
- "我这里老是湿湿的"回到个人痛点，让推荐更可信。

### 可复用公式

```
我一般[使用时机]涂，到[结束场景]之前，[部位状态]都还比较[效果]。
```

### 合规表达建议

建议用"比较干爽""没有明显味道""我自己用下来"这类体验表达，少用绝对化表述。

## 六、个人习惯背书拆解

### 原文

```
我的腋下是比较爱出汗，所以一年四季都会用这些产品。
```

### 作用

- 建立"长期使用者"身份。
- 解释为什么自己有发言权。
- 为后面的竞品对比做铺垫。

### 可复用公式

```
我自己属于[长期痛点]，所以这类产品我真的用过不少。
```

## 七、竞品对比拆解

### 原文

```
我以前用过滚珠的，还有膏体的，但是我觉得夏天穿衣服就喜欢穿紧身的，用那种就超级黏，但是它这个就完全不会。
```

### 作用

- 提出使用经验：滚珠、膏体都用过。
- 说出旧产品的问题：夏天穿紧身衣会黏。
- 突出本产品优势：不黏。
- 场景非常具体，容易引发共鸣。

### 可复用公式

```
我以前用过[竞品类型]，但在[具体场景]会[不适感]，这个让我喜欢的是[优势]。
```

### 示例

```
我以前用滚珠会觉得黏，尤其穿紧身衣的时候很明显，但这个乳霜吸收后就舒服很多。
```

## 八、成分背书拆解

### 原文

```
它里面的氯化羟铝就是专门针对这一块环境的，真的很清爽，而且很持久。
```

### 作用

- 用成分增加可信度。
- "这一块环境"用口语化方式避开复杂解释。
- 成分后立刻回到体感：清爽、持久。

### 可复用公式

```
它里面有[成分/技术点]，比较适合[场景/部位环境]，用下来就是[体感结果]。
```

### 注意

成分表达建议保持克制，避免医疗化、治疗化承诺。

## 九、价格利益点拆解

### 原文

```
主要是它也很便宜，我这一支用了有一个多月，现在还剩一点，直接去囤个三支都够用一整个夏天了。
```

### 作用

- "便宜"降低下单阻力。
- "一支用了一个多月"证明用量省。
- "三支够用一整个夏天"给出购买理由。
- 适合带出套装规格。

### 可复用公式

```
一支我用了[使用周期]，现在还剩[剩余情况]，所以[规格/套装]基本够[使用周期]。
```

## 十、结尾转化拆解

### 原文

```
只要是这里爱出汗的，这个就真的可以试一下，到手就是这样子的三支。
```

### 作用

- 再次点名适合人群。
- "可以试一下"转化语气自然，不强硬。
- "到手三支"展示规格和获得感。

### 可复用公式

```
如果你也是[痛点人群]，这个真的可以试一下，到手是[规格/数量]。
```

## 完整脚本重组模板

```
A：你是不是也这样？明明洗过腋下，但出门没多久还是有味。
B：这种真的别只靠洗，出门前可以先涂这个。

我今天早上出门前涂的，现在已经下午两点了，腋下还是比较干爽。
它上面写的是止汗净味，像我这种腋下爱出汗的人，夏天会很需要。

它是乳霜质地，涂开几秒就吸收了。
不会黏，也没有什么不舒服，涂完之后穿衣服也不会觉得糊着。

我以前用过滚珠，也用过膏体。
但是夏天穿紧身衣的时候，有些真的会黏。
这个我比较喜欢的一点就是清爽。

里面有氯化羟铝，适合腋下这种容易出汗、容易潮湿的局部环境。
我一般出门前15分钟涂，到晚上洗澡之前，都没有明显味道。

而且它一支能用挺久，我这一支用了一个多月，现在还剩一点。
腋下爱出汗、怕味道尴尬的，可以直接试一下。
到手是这种三支装，夏天囤着刚好。
```

## 分镜建议

| 段落 | 画面 | 口播重点 |
| --- | --- | --- |
| 0-3秒 | 两人对话/捂腋下尴尬表情 | 洗了还是有味怎么办 |
| 3-8秒 | 拿出产品 | 出门前涂止汗净味乳霜 |
| 8-15秒 | 展示时间/通勤/下午状态 | 早上涂，下午仍然干爽 |
| 15-25秒 | 手背试涂 | 乳霜质地，几秒吸收 |
| 25-35秒 | 穿紧身衣/抬手动作 | 不黏、不闷、不尴尬 |
| 35-45秒 | 成分和包装特写 | 氯化羟铝，清爽持久 |
| 45-55秒 | 三支装展示 | 便宜、耐用、适合囤 |

## 拍摄注意

- 前3秒必须有痛点，不要先介绍品牌。
- 产品出现后马上说使用场景：出门前15分钟。
- 体验表达要具体：几点涂、几点验证、穿什么衣服、什么感受。
- 成分只做辅助背书，不要讲成专业讲座。
- 结尾用"适合谁"收口，比单纯喊下单更自然。
- 避免绝对化词：永久、根治、100%、彻底、保证无味。



# 格式二：
# 抖音口播脚本模板（40-60秒）

```
[0-3秒 钩子]
[人群/场景] + [反常识或痛点提问]

[3-12秒 痛点炸裂]
你有没有这种感觉：[具体场景，断句+感叹号]
这！不！是！XX！

[12-25秒 核心观点]
[颠覆认知陈述]
[简化机制解释，一两句]

[25-38秒 数据/证明]
[具体数字] + [时间单位] + [反差结论]

[38-50秒 行动建议]
怎么办？就[X]件事：
第一……
第二……

[50-60秒 互动收割]
[提问引导评论] 👇
```

## 检查清单
- [ ] 前3秒出现结论或强烈钩子
- [ ] 每30秒至少1次情绪爆发
- [ ] 平缓段落不超过10秒
- [ ] 结尾有明确互动引导
