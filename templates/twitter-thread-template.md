# Twitter Thread模板（8-10条）

```
Tweet 1【Hook】
hot take: [争议性观点]
and nobody talks about this. 🧵

Tweet 2【Context】
[建立可信度的背景，像真人在说话]

Tweet 3-7【核心论点，每条一个】
[观点] + [解释] + [数据/例子]
结尾用cliffhanger：but here's where it got interesting...

Tweet 8【Twist】
[反转或最意外发现，这条最容易被单独转发]

Tweet 9-10【CTA】
[互动引导]
RT the first tweet so others can see it.
```

## 语言规则
- 不完整句，省略主语
- 停顿技巧：every. single. day. / it just... disappears.
- 单字成句：yeah. / nope. / listen.
- ❌ This is amazing → ✅ ok this actually blew my mind
