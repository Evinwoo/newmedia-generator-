# 小红书图文模板

## 标题公式
[人群词] + [症状/场景词] + [数量词] + [效果词]
示例：30+女性 | 吃XX一个月 | 真实变化记录

## 正文结构（体验文）
```
开头钩子（场景/情绪，100-300字）
↓
自然过渡："后来朋友/营养师推荐了XX"
↓
产品介绍（用用户语言，不超过3句）
↓
效果描述（具体场景化，避免空泛词）
↓
结尾：总结+信任背书（复购/推荐给朋友）
```

## 评论区预埋
- 「我XX情况可以用吗？」
- 「需要用多久才有效果？」
- 「在哪里买的？」

## 排版检查清单
- [ ] 每段不超过5行
- [ ] 重点单独成行
- [ ] emoji适量不过度
- [ ] 字数符合类型要求（体验文精简/攻略文可达1000字）
