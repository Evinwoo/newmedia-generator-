# 知乎平台 Skill

> 核心定位：专业感 + 信任感 + 搜索长尾收割
> 知乎是搜出来的，不是刷出来的

---

## 一、内容逻辑

```
用户搜索"XX有什么副作用"
→ 看到你的高赞回答
→ 建立信任
→ 点进主页或评论区找购买渠道
```

---

## 二、标题/问题选择

| 问题类型 | 示例 | 转化潜力 |
|----------|------|----------|
| 选购类 | XX和XX哪个更值得买？ | ⭐⭐⭐⭐⭐ |
| 避坑类 | 买XX要注意什么？ | ⭐⭐⭐⭐ |
| 经历类 | 吃了XX之后有什么感受？ | ⭐⭐⭐⭐ |
| 原理类 | XX为什么有效/无效？ | ⭐⭐⭐ |

---

## 三、正文结构

### 科普解析文
```
开头：直接给出核心结论（结论前置）
→ 问题定义：为什么这件事值得认真对待
→ 原理拆解：数据/研究/成分说话
→ 市面现状：选购时常见的坑
→ 我的选择：自然植入产品推荐
→ 结尾：总结+行动建议
```

### 亲身经历文
```
开头：我的具体问题（时间+症状+数量词）
→ 探索过程：踩过哪些坑
→ 转折：发现了这个产品/方法
→ 效果：具体数据/对比
→ 结尾：给有同样问题的人的建议
```

---

## 四、写作规则

| 维度 | 要求 |
|------|------|
| 专业度 | 必须引用真实数据、研究、成分名称 |
| 可读性 | 长文必须分小标题，每节不超过300字 |
| 态度 | 客观理性，适当承认局限性（反而增加信任） |
| 植入方式 | 产品出现在"经过筛选后的自然选择"环节 |

---

## 五、格式规则

- 善用**加粗**标注关键词和数据
- 分级标题（## ###）
- 关键对比用**表格**呈现
- 全文建议2000-4000字
- 评论区置顶补充购买渠道（不在正文放链接）
