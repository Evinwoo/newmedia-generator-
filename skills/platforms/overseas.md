# 海外平台 Skill — Twitter/X · Instagram · Reddit

> 核心原则：不是翻译，是用英文母语者的思维方式重新表达

---

## 一、三平台用户画像

| 维度 | Twitter/X | Instagram | Reddit |
|------|-----------|-----------|--------|
| 用户心态 | 观点碰撞，求新求快 | 视觉种草，生活方式认同 | 真实体验，反广告敏感 |
| 对品牌态度 | 接受，但要有料 | 接受，但要美 | 极度反感硬广 |
| 转化路径 | 点bio链接 | 点主页链接 | 评论区自然引导 |

---

## 二、Twitter/X 规则

### Thread结构
```
Tweet 1【Hook】
hot take: [争议性观点]
and nobody talks about this. 🧵

Tweet 2【Context】建立可信度的背景

Tweet 3-8【每条一个论点】
观点 + 解释 + 数据
结尾用cliffhanger过渡

Tweet 9【Twist】最意外的发现

Tweet 10【CTA】
互动引导 + 转发引导
```

### 语言风格

| ❌ 不要 | ✅ 要 |
|--------|-------|
| This product is amazing | Took it for 3 weeks. My inflammation dropped. |
| Studies show that | here's what the research actually found |
| Please follow | bookmark this. you'll want it later |
| 翻译腔完整句子 | 省略主语的不完整句 |

- 用缩写：you're / I've / doesn't
- 用单字成句：yeah. / nope. / listen.
- 用停顿：every. single. day. / it just... disappears.

## 三、Instagram 规则

### Reels
- 前3秒：直接说结论或问题，不要logo/intro
- 字幕必须跟口播同步（大量用户静音观看）
- 结尾：明确CTA "Link in bio" 或 "Save this"

### Carousel（图文轮播）
```
第1张：大字标题抓眼球
第2-7张：每张一个论点，左上角编号
最后一张：总结 + "Save this post"
```

### Caption
```
第一行（125字符内）：Hook句
正文：真实故事/具体价值，换行和emoji分隔
结尾：一个问题引导评论
Hashtag：3-10个，放在最后
```

---

## 四、Reddit 规则

> ⚠️ Reddit极度反广告，硬广会被downvote甚至ban

### 帖子写法
```
标题：具体+数字+真实感
✅ "Took fish oil daily for 90 days. Here's what changed"
❌ "Amazing fish oil that transformed my skin!"

正文：先讲背景，承认局限性，产品自然植入
"The one I ended up using was [产品名] — not sponsoring, just what worked"

结尾：邀请讨论，不要CTA
```

### 操作规则
- 发帖后第一小时内积极回复每条评论
- 不要在帖子里放购买链接
- 遇到质疑不删，认真回复反而增加可信度
