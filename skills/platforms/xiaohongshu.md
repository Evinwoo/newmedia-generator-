# 小红书平台 Skill

> 适用场景：小红书图文笔记
> 核心目标：推送给正确人群 + 实现商业转化，不以追求爆文为唯一目标

---

## 一、标题规则

### 1.1 标题构成要素

| 元素 | 说明 | 举例 |
|------|------|------|
| 人群词 | 锚定目标受众 | 黄黑皮、30+女性、考研党、宝妈 |
| 症状词 | 描述用户痛点 | 脱发、肝损伤、注意力不集中 |
| 热点词 | 蹭当前搜索热度 | 维A酸、辅酶Q10、意大利药店 |
| 数量词 | 增加真实感 | 吃了1周、30天、才十几块 |
| 品类词 | 锁定搜索意图 | 鱼油、维生素D3、保健品 |
| 效果词 | 直接呈现结果 | 震惊了、真的媲美、反馈 |

### 1.2 标题目标分类

**转化型**（锁定已有购买意向，转化率高但流量小）
- 示例：`金凯撒鱼油吃了1周反馈`

**拉新型**（扩大人群认知）

| 类型 | 示例 |
|------|------|
| 效果吸引 | 黄黑皮吃鱼油30天真的震惊了！ |
| 恐吓吸引 | 不要再吃鱼油了小心肝损伤！ |
| 海外背书 | 大家一定要去买意大利药店的这款鱼油 |
| 类目对比 | 这个东西真的媲美维A酸软膏！ |
| 年龄认同 | 30+女性到底该吃什么保健品？ |

## 二、开头钩子规则

> 核心原则：场景/情绪类可100-300字，其他类型控制在150字以内

### 类型一：颠覆性观点
- 必须基于真实研究或近期平台搜索量上升的话题
- 示例：`你以前的维生素D3都吃错了！`

### 类型二：具体场景（重点）
- ❌ 错误：堆砌多个场景（经常吃外卖、熬夜、压力大）
- ✅ 正确：选定一个场景深挖，大量使用数量词，还原真实感
- 示例：`马上要12月份考研了，我从10月份开始就发现发根开始有松动！洗头的时候掉的多大概就是五六十根，现在每天早晚梳头密密麻麻的一百多根，我真的哭了`

### 类型三：具体情绪
- 直接口语化表达，不需要科普
- 示例：`强推一个无人问津的才十几块！爱上洗手了555`

### 类型四：方法论
- 适用于攻略类，先总结后发散

### 类型五：权威背书
- 用权威人物/机构的原话直接开头

### 类型六：社会热点
- 将产品功效与当前热点事件关联

## 三、正文结构规则

### 测评文
```
开头钩子 → 定义好产品标准 → 对比多款产品（自家只说优点，竞品抓缺点避雷）→ 最终推荐
```

### 广告文
```
开头钩子 → 产品为什么能解决问题 → 为什么选我家 → 优惠机制/使用感受
```

### 体验文（软广）
```
具体场景深挖细节和情绪 → 中间自然植入产品 → 为什么买（朋友/营养师推荐）→ 产品效果（用用户语言，不过度介绍）
```

### 攻略文
```
开头钩子 → 哪些方法可以解决问题 → 推荐产品
```

## 四、结尾与评论区

### 结尾（三选一或组合）
- 直接总结产品好处
- 信任背书：已推荐给X个朋友/已复购X次
- 优惠逼单：当前活动价/限时优惠

### 评论区引导

| 类型 | 目的 | 示例 |
|------|------|------|
| 对号入座型 | 引导互动 | 我有XX问题，可以用这款吗？ |
| 方法论型 | 促进信任 | 我XX问题需要用多久？ |
| 收割型 | 引导购买 | 在哪里买？ |

---

## 五、排版规则

- 一段话/一个重点单独一行，方便扫读
- 适当添加匹配内容的 emoji
- 除真实体验文外，每段不超过5行
- 字数：攻略文可用满1000字，其他类型讲清楚即可

---

## 六、标签选择规则

1. 从标题和正文提取核心关键词
2. 在小红书搜索，优先选该词下热度最高的话题
3. 搭配：大热度话题2-3个 + 精准垂类3-5个 + 品牌词1-2个

---

## 七、与抖音互转规则

| 维度 | 小红书 | 转抖音后 |
|------|--------|----------|
| 文字量 | 可以较多 | 大幅压缩 |
| 信息顺序 | 钩子→铺垫→结论 | 结果和重点必须前置 |
| 语言风格 | 小红书人群语境 | 换成抖音人群语境 |

> 小红书是"讲故事"，抖音是"给答案"
