# 平台转化提示词

## 单平台转化

```
你是一个专业的跨平台内容转化专家。

请将以下原始内容转化为【目标平台】的适配版本，目标人群是【人群】。

【原始内容】
（粘贴原文）

【转化规则】
- 保留原文核心卖点，不添加原文没有的产品信息
- 严格遵守目标平台的写作规则
- 用目标人群最熟悉的语言和场景表达
- 直接输出内容，不需要解释转化过程
```

## 小红书 → 抖音

```
请将以下小红书图文转化为抖音口播脚本（40-60秒）。

【原文】（粘贴）

【规则】
1. 时长40-60秒（约200-300字）
2. 结构：[0-3秒钩子]→[3-12秒痛点]→[12-25秒核心]→[25-38秒证明]→[38-50秒互动]
3. 情绪密度：每30秒至少1次爆发点
4. 结论必须在前10秒内出现
5. 结尾有互动引导（扣数字/@好友）

输出格式用时间轴标注每个段落。
```

## 小红书 → 知乎

```
请将以下小红书内容扩写为知乎长文（2000-4000字）。

【原文】（粘贴）
【参考研究资料】（如有，粘贴research文件夹中相关内容）

【规则】
1. 结论前置，开头直接给答案
2. 引用真实数据/研究支撑核心观点
3. 客观理性语气，以过来人口吻
4. 产品自然植入在"经过筛选后的选择"环节
5. 善用加粗、分级标题、表格
```

## 一键多平台

```
请将以下原始内容同时转化为多个平台版本。

【原始内容】（粘贴）

请依次输出：
1. 抖音口播脚本（40-60秒）
2. 知乎长文（结论前置，引用数据）
3. Twitter Thread（英文，8-10条，hot take开场）

每个版本之间用「---」分隔。
```
