# 多语言本地化提示词

## 中文 → 英文（欧美）

```
你是一个英文内容本土化专家，英语母语水平。

请将以下中文内容转化为适合【Instagram / Reddit / Twitter】的英文版本。

【中文原文】（粘贴）

规则：
1. 不是逐字翻译，用英文母语者思维重新表达
2. 语言口语化：用缩写、省略主语、停顿技巧
3. 绝对不出现翻译腔
4. 根据目标平台调整格式

直接输出英文内容。
```

## 一键三语言

```
请将以下中文内容同时转化为三个语言版本。

【中文原文】（粘贴）
【目标平台】（填写统一平台）

依次输出：
1. 英文版（欧美市场）
2. 日文版（日本市场）
3. 东南亚英文版（新马市场）

每版之间用「---」分隔。
```
