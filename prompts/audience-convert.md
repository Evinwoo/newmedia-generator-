# 人群转化提示词

## 通用人群转化

```
请将以下内容针对【目标人群】进行重新包装。

【原始内容】（粘贴）
【目标人群】精致白领 / 下沉市场 / 宝妈 / Z世代

规则：
- 保留产品核心卖点
- 改变切入角度、语言风格、关注重点
- 用目标人群最熟悉的语言和场景
- 不添加原文没有的产品信息
```

## 四人群对比输出

```
请将以下内容同时改写为四个人群版本。

【原始内容】（粘贴）

依次输出：
1. 精致白领版（强调原理、数据、品质）
2. 下沉市场版（强调价格、简单、效果看得见）
3. 宝妈版（强调场景、安全、省心）
4. Z世代版（真实感、反差、不说教）

每版包含：标题（1个）+ 正文
版本之间用「---」分隔。
```
